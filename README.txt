CERTIFICATES
The Certificates page contains the professional credentials listed in the CV and is designed for easy extension. To add future certificates, copy one <article class="certificate-card"> block in certificates.html and update the title, issuing body, year, credential details and verification URL. Certificate images or PDFs can be placed in assets/certificates/ and linked from the relevant card.

DESIGN
The entire site now uses a shared modern visual system inspired by the approved reference design: dark navy gradient hero sections, teal/cyan accents, glass-style elements, rounded cards, modern typography, responsive navigation and subtle reveal animation.
